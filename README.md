# Swag Labs (SauceDemo) QA Testing Portfolio

**Project built to demonstrate QA skills for QA Trainee role**  
**Background:** Data Science / AI-ML → transitioning to QA

### Project Highlights
- 12 Manual Test Cases covering Login, Products, Cart, Checkout & Logout
- Automated tests using Python + Selenium + Pytest
- Follows STLC (Software Testing Life Cycle)
- Includes test documentation, bug report template & execution report

**Technologies:** Python, Selenium WebDriver, Pytest, GitHub

**GitHub Repo:** https://github.com/kabsth/swaglabs-qa-testing-portfolio

### Skills Demonstrated
- Test Case Design (Positive + Negative + Edge cases)
- Functional & End-to-End Testing
- Automation Framework basics
- Defect/Bug Reporting
- Test Reporting

Made quickly as a self-learning project to show initiative and fast learning ability.

### How to Run (for recruiters)
```bash
pip install -r requirements.txt
pytest automation/ --html=reports/test_report.html --self-contained-html
